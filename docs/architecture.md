# Architecture

## Overview
The architecture of ArcTracker v2 is built around a modular design that allows for scalability and maintainability.

## Components
- **Client Application**: The frontend interface for users.
- **API Server**: Handles requests from the client and serves data.
- **Database**: Stores user data and application state.
